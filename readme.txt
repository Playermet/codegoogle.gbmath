The math library for the programming language c + +. 
Easy to use, to implement the graphics and entertainment applications.

Contains classes for mistakes, as in two-and three-dimensional space. 
Search for intersections and collisions. 
Integration with popular physics engine, such as: box2d, Newtone, etc. 
All classes have assignment operators to / from a string to facilitate serialization.
